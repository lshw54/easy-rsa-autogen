# easy-rsa-autogen
One-click certificate generation for easy-rsa

## How to use
```bash
mkdir -p autogen && cd autogen
wget -O - https://raw.githubusercontent.com/lshw54/easy-rsa-autogen/main/easyrsa.sh | bash
```
